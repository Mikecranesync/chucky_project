# Claude Code Sub-Agents

This directory contains specialized sub-agent definitions for automating Chucky workflow development tasks.

## Available Agents

- **n8n-workflow-architect** - Generates complete n8n workflows from requirements
- **industrial-knowledge-curator** - Scrapes and formats industrial documentation for RAG
- **discord-integration-specialist** - Configures Discord nodes and message routing
- **documentation-maintainer** - Auto-updates setup guides and READMEs
- **testing-coordinator** - Creates test plans and validation scripts

## How Agents Work

Sub-agents are autonomous components that can be invoked by slash commands to handle complex, specialized tasks. Each agent has specific domain expertise and follows defined patterns for output generation.
